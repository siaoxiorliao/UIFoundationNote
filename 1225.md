## storyboard控件连线

* 控件只有继承自UIControl才能有连线时间,比如Label就没有</br>
* sender:拿到控件
* 经典错误:
>* 错误一: this class is not key value coding-compliant for the key testLabel</br>
原因:有多余的连线</br>
解决:查看控件然后删除多余的连线
>* 错误二: unrecognized selector sent to instance 0x7feb69418640</br>
原因:找不到对应的方法</br>
解决:1.添加对应的方法 2.删除多余的连线
